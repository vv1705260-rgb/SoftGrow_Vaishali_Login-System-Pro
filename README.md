Python Task 1 - Project 2 for SoftGrowTech Internship

## 📋 Description
A secure command-line Login & Signup System built with Python. It handles user registration, authentication, and admin view with extra security features.

This project demonstrates file handling, user input validation, and basic security practices in Python.

## ✨ Special Features
1. **Hidden Password Input** - Uses `getpass` module so passwords don't show while typing
2. **Password Strength Check** - Enforces 6+ characters with letters + numbers
3. **Duplicate Username Check** - Prevents multiple accounts with same username
4. **Login Attempt Limit** - Locks after 3 failed attempts for security
5. **Signup Date Tracking** - Records and displays account creation date
6. **Admin View** - Option to view all registered users with join dates

## 🛠️ Tech Used
- **Language:** Python 3
- **Modules:** `getpass`, `datetime`, `os` - all built-in, no installation needed
- **Storage:** `.txt` file for user data
